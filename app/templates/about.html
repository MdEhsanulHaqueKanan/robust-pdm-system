About This Project

Robust and Interpretable Predictive Maintenance for Evolving Industrial Systems

This is an independent research and development project by Md. Ehsanul Haque Kanan, created to showcase an end-to-end, industry-standard approach to predictive maintenance.

Core Objectives:

  • Predictive Power: To accurately predict both Remaining Useful Life (RUL) for degrading systems and specific fault types for event-based failures.
  • Interpretability (XAI): To move beyond "black box" models by providing clear, human-understandable explanations for each prediction using SHAP (SHapley Additive exPlanations).
  • Robustness & Adaptivity: To demonstrate how a system can monitor model performance and detect concept drift, which is critical for real-world deployment in "evolving" industrial environments.

Technology Stack:

  • Backend: Python, Flask
  • Machine Learning: Scikit-learn, XGBoost, LightGBM, SHAP, Imbalanced-learn
  • Data Handling: Pandas, NumPy
  • Frontend: HTML, Pico.css, Plotly.js
  • Deployment: Docker (to be implemented)
